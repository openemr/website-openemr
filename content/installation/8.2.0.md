# Installing OpenEMR 8.2.0

## Prerequisites

- PHP 8.2 or newer with the OpenEMR-required extensions installed.
- MySQL 5.7+ or MariaDB 10.4+.
- Apache 2.4+ or Nginx 1.18+ in front of PHP-FPM.
- A modern web browser for the setup wizard.

## Download

Download the OpenEMR 8.2.0 release tarball:

- [openemr-8.2.0.tar.gz](https://github.com/openemr/openemr/releases/download/v8_2_0/openemr-8.2.0.tar.gz)

## Install

1. Extract the tarball into your web server's document root.
2. Set file permissions according to the OpenEMR installation guide.
3. Browse to `https://your-server/openemr/setup.php` and follow the wizard.
4. Complete database setup, admin account creation, and initial configuration.

## After install

- Review the [administrator manual](/docs/administration/) for first-time
  configuration.
- Apply the [security checklist](/docs/security/) before exposing the
  installation publicly.
